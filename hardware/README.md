# Аппаратная часть

Минимальный состав изделия:

- головная плата на `nRF52840`;
- аналоговый тракт емкостного датчика;
- датчик температуры топлива;
- датчик температуры воздуха;
- сервисный `USB` порт;
- защита питания и входов;
- герметичная головная часть;
- погружной зонд.

## База для прототипа

Для первых образцов можно не делать сразу собственную цифровую плату, а использовать одну из готовых:

- `Promicro NRF52840 Development Board Compatible with nice!nano V2.0`
- `Super 52840 NRF52840 Development Board Compatible with Seeed XIAO BLE`

В этом случае аппаратная часть делится на:

- модуль управления на готовой dev-board;
- отдельную плату измерительного фронтенда и питания;
- механическую часть зонда и головной узел.
