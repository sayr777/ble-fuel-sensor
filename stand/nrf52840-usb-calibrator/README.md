# Стенд тарировки nRF52840 USB

Назначение стенда:

- проверка связи с датчиком по `USB`;
- запуск сервисной тарировки;
- чтение сырых измерений;
- проверка BLE-публикации;
- контроль записи конфигурации в память.

Минимальный состав:

- хост-ПК или ноутбук;
- сервисное ПО тарировки;
- эталонный датчик или имитатор емкостного канала;
- BLE-приемник или телематический блок для контрольного сценария.

## Вариант стенда на прототипах `ProMicro NRF52840`

Для быстрого старта стенд можно собрать без собственной цифровой платы, используя готовые `Promicro NRF52840 Development Board Compatible with nice!nano V2.0`.

Рекомендуемая минимальная конфигурация:

- `ProMicro #1` - прототип `ДУТ`;
- `ProMicro #2` - эмулятор `БНСО` или BLE-приемник;
- ПК по `USB` для сервисной настройки, журналирования и обновления прошивки;
- внешний макет измерительного канала, датчиков температуры и питания.

- [docs/promicro-prototype-test-method.md](docs/promicro-prototype-test-method.md)
