# Внешние спецификации

В этот раздел стоит складывать локальные копии или ссылки на:

- datasheet `nRF52840`;
- pinout и схему `Promicro NRF52840 compatible with nice!nano v2`;
- pinout и схему `Super 52840 compatible with Seeed XIAO BLE`;
- документацию по `USB` защите и питанию;
- спецификации датчиков температуры;
- требования к материалам и уплотнениям для работы с топливом;
- согласованный формат обмена с телематическим блоком;
- требования `РНИС` к полезной нагрузке данных.
